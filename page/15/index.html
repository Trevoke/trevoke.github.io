I'm taking this from this page but thought I should try to propagate it a little since it took me forever to find it. Assume in this example that the username is 'atiensivu', the domain is 'staff', and the user profile is located in %systemdrive%\users\atiensivu. %systemdrive% is typically C: on standard Vista installs. Replace every instance 'atiensivu' with the username of the corrupt profile. If you can log into the profile, make note of the %USERPROFILE% environment variable value.
